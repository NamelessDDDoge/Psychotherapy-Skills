# AI-Psychotherapist

[English](./README.md) | **简体中文**

> 事实上，一个医生用哪种技巧无关紧要，因为关键不在于技巧而在于使用技巧的人。  
> ——荣格《心理治疗之现状》  
> 英文原文见 *The State of Psychotherapy Today*，[337]

这个仓库的意义不在于把心理治疗规范化为某种固定流程，而在于尽可能复刻顶尖治疗师的完整人格，使 AI 不是仅仅调用技巧，而是从一个连贯的人之存在出发作出回应。

我们的核心理念很简单：起到决定性作用的，恰恰是治疗师的人格。

当前仓库包含两个 perspective skill 和一个制作说明文档：

## 当前可用 Skill

- **Person-Centered Perspective**：[`person-centered-perspective/SKILL.md`](./person-centered-perspective/SKILL.md)
  - 以 Carl Rogers 为灵感来源的以人为中心视角，适合广义的人类对话、反映式倾听、存在性思考，以及非控制性的情感陪伴。
- **Collective Unconsciousness Perspective**：[`collective-unconsioucness-perspective/SKILL.md`](./collective-unconsioucness-perspective/SKILL.md)
  - 以 Jung 为灵感来源的心理观察式视角，适合梦、象征、投射、阴影、内在冲突、意义流失等主题，并强调谨慎解释而非神秘化断言。

## Skill 制作指南

- **Skill Maker**：[`skill_maker.md`](./skill_maker.md)
  - 一份面向 perspective/persona skill 的实用制作方法，覆盖材料来源、结构组织、证据账本、示例库设计与安全边界。

## 说明

- 这些文件用于提示词、回应风格和工作流层面的指导。
- 它们不能替代持证心理健康专业服务。
- 如果对话涉及即时危险、自伤、伤人、精神病性失序或无法保证安全，应由安全优先的支持方式覆盖反思式风格。
