# Psychotherapy Skills

**English** | [简体中文](./README_CN.md)

A repository of psychotherapy-style skills for AI role, tone, and response-process design.

This repo currently contains two perspective skills and one writing guide:

## Available Skills

- **Person-Centered Perspective**: [`person-centered-perspective/SKILL.md`](./person-centered-perspective/SKILL.md)
  - A Carl Rogers-inspired, person-centered stance for broad human conversation, reflective listening, existential reflection, and non-coercive emotional presence.
- **Collective Unconsciousness Perspective**: [`collective-unconsioucness-perspective/SKILL.md`](./collective-unconsioucness-perspective/SKILL.md)
  - A Jung-inspired, psychologically observational stance for dreams, symbols, projection, shadow, inner conflict, meaning-loss, and symbolic interpretation without mystical overreach.

## Skill Construction Guide

- **Skill Maker**: [`skill_maker.md`](./skill_maker.md)
  - A practical guide for building high-quality perspective/persona skills, including source selection, structure, evidence handling, example-bank design, and safety boundaries.

## Notes

- These files are for prompting, response-style, and workflow guidance.
- They are not a substitute for licensed mental health care.
- If a conversation involves immediate risk, self-harm, harm to others, psychosis, or inability to stay safe, safety-first support should override the reflective style.
